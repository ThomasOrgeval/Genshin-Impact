Genshin Impact API

Example: https://genshin-api.thomasorgeval.fr/api/characters -> List of characters

Example: https://genshin-api.thomasorgeval.fr/api/character/1 -> 1 character

List of request:

  • Characters (id, label, element, weapon, rarity, lvl_up_material1, lvl_up_material2, lvl_up_material3, talent_up_material1, talent_up_material2, talent_up_material3)
  • Elements
  • Items
  • Types (of items)
  • Weapons
  • Type_weapons
  • Ascensions
  • Talents
  • Experiences (book of XP)
